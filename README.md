## ✨ Features
- 🤖 AI/ML Stock Price Prediction (Linear Regression + Random Forest)
- 📊 Real-time Stock Dashboard with Charts
- 🔮 7–30 Day Price Forecasting
- 🤖 AI Chatbot (NLP-powered)
- 💼 Portfolio Tracker with P&L
- 🌍 Market Overview (S&P 500, NASDAQ, Dow)
- 📈 Live Stock Ticker
- 📰 Market News Feed

## 🧠 ML Pipeline
1. Data: Yahoo Finance API (2 years)
2. Features: 18 technical indicators (MA, RSI, Bollinger Bands, etc.)
3. Models: Linear Regression (40%) + Random Forest (60%)
4. Metrics: MAE, RMSE, R² Score
