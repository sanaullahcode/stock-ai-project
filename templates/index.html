📐 About the Model

Our ensemble model combines Linear Regression (40% weight) and Random Forest (60% weight) trained on 18 technical features including Moving Averages (5/10/20/50-day), RSI, Bollinger Bands, Volume ratios, and momentum indicators. Training uses 80% historical data with 20% test split.

🏗️ System Architecture
🧠 ML Pipeline

1. Data Fetch — Yahoo Finance API (2 years)
2. Feature Engineering — 18 technical indicators
3. Model Training — Linear Regression + Random Forest
4. Ensemble — 40% LR + 60% RF weighted average
5. Evaluation — MAE, RMSE, R² Score
6. Prediction — Next N days rolling forecast

⚙️ Tech Stack

Backend: Python 3.11, Flask, Flask-CORS
ML: scikit-learn, NumPy, Pandas
Data: yfinance (Yahoo Finance API)
Frontend: HTML5, CSS3, JavaScript ES6+
Charts: Chart.js 4.4
